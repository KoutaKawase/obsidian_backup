- ダライ・ラマですら怒りの感情を持つことはある => でも鎮め方を心得ている

```
彼のような人徳者であっても、常にブッダでいることはできません。人にやさしく接するなんてできないというときは、誰にでもあります。私たち禅僧は、坐禅で心を整えます。みなさんは、そういうときは無理をしすぎず、「パートタイム・ブッダ」を心がけましょう。
```

```
大切なのは、その怒りや不安で揺らいだ心を穏やかにする自分なりの方法を知ることです。
```
