## 余計なことを言うな
```
**コミュ障とは極端に我慢が出来ない人のことで  
（ネガティブな内心全部すぐ顔や態度に出る）  
黙ってても警戒心や陰湿な陽キャ嫌悪オーラをだだ漏れにして嫌われるので  
「大人しくて無害な良い人と思われる」ポジションは  
決して陰気なだけのコミュ障ではなれないんです。**
```

```
**真のコミュ障とは無口な人でなはく、むしろ逆で  
隙あらば取らなくてよいマウントを取っていらんことをベラベラ喋るやつの事なんすよ。  
職場にも高校までは進学校だったけどいじめで没落した奴が  
聞かれもしないのに高校の名前出して「すごいね」待ちするから  
うざがられてハブられてる。**
```
